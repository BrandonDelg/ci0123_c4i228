# Simulación de Packet Tracer

## Descripción

La simulación usa un switch 2960 para conectar un cliente, un servidor tenedor, y un servidor de piezas. Se configuran las interfaces fastEthernet de la 0/1 a la 0/12 con la vlan 250(Asignada a Equipo 5), a la que se le asigna la red 172.16.10.0/24(De prueba, en el enunciado no indica). A cada dispositivo final conectado se le asigna una ip manual de la siguiente manera:

- Cliente: 172.16.10.10
- Servidor Tenedor: 172.16.10.11
- Servidor Piezas: 172.16.10.12
