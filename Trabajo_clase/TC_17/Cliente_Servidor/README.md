# Cliente-Servidor
## Instrucciones de compilación
```
# General
make clean
make

# Servidor
make server

# Cliente
make client

# Filesystem
make insertarfs

```
## Instrucciones de ejecución
```
# Ejecución de servidor
make run-server ARGS="<host> <ipv6 0|1> <puerto>"

# Ejecución del cliente
make run-client ARGS="<host> <ipv6 0|1> <puerto>"

# Ejecucion del filesystem
make run-insertarfs

```
